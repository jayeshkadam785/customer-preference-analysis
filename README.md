# 🍽️ Customer Preference Analysis

A data analysis project completed as part of **Cognifyz Technologies – Data Analytics Internship (Level 3, Task 2)**.  
This project explores customer preferences by analyzing the relationship between restaurant cuisines, ratings, and customer votes.

---

## 📌 Project Objective

The main objective of this project is to:

- Analyze the relationship between **cuisine type** and **restaurant ratings**
- Identify the **most popular cuisines** based on customer votes
- Determine whether certain cuisines tend to receive **higher ratings**
- Generate insights that can help restaurants improve customer satisfaction

---

## 📂 Dataset Information

The dataset contains restaurant-related information such as:

- Restaurant Name
- Cuisine Type
- City
- Aggregate Rating
- Number of Votes
- Price Range
- Other restaurant features

---

## 🔍 Tasks Performed

### 1. Cuisine vs Rating Analysis
- Calculated the average rating for each cuisine
- Compared ratings across different cuisine categories

### 2. Popular Cuisine Analysis
- Aggregated customer votes by cuisine
- Identified the cuisines with the highest customer engagement

### 3. High-Rated Cuisine Identification
- Filtered cuisines with consistently high average ratings
- Highlighted cuisines preferred by customers
